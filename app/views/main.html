angular-better-placeholders

Better, more friendly implementation of placeholder for form fields inside of Bootstrap. and elsewhere
Inspired by a really cool Dribble concept.

Demo

Begin typing in a field to trigger the placeholder text. If an input ever becomes unfocused and is empty, the placeholder should disapear.

Usage

Simple include the betterPlaceholder directive as a class on your input fields.

<input type="text" ng-model="user.firstName" class="form-control better-placeholder" placeholder="First Name" />

Include the following CSS snipets:

.better-placeholder {
  transition: all .2s ease;
}
.better-placeholder.better-placeholder-active {
  padding: 34px 12px 16px 12px;
}
.better-placeholder-text {
  position: absolute;
  top: 5px; left: 28px;
  font-size: 10px;
  font-weight: bold;
}
